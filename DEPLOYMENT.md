# 部署指南

## 快速部署

### 1. 构建生产版本
```bash
npm run build:prod
```

### 2. 部署选项

#### A. Netlify (推荐)
1. 将项目推送到 GitHub
2. 在 Netlify 中连接仓库
3. 构建命令：`npm run build:prod`
4. 发布目录：`dist`

#### B. Vercel
1. 安装 Vercel CLI：`npm i -g vercel`
2. 运行：`vercel --prod`

#### C. 静态服务器
```bash
# 构建后直接部署 dist/ 文件夹到任何静态服务器
npm run build:prod
# 将 dist/ 文件夹上传到服务器
```

## 长期维护建议

### 1. 数据备份策略
- 定期使用导出功能备份数据
- 建议每周导出一次作为备份

### 2. 依赖更新
```bash
# 检查过时的依赖
npm outdated

# 更新依赖（谨慎操作）
npm update
```

### 3. 监控和日志
- 使用浏览器开发者工具监控错误
- 定期检查控制台日志

## 已知限制

1. **数据存储**：仅支持单浏览器存储
2. **容量限制**：约 5MB 数据上限
3. **协作功能**：暂不支持多用户
