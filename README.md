# 提示词管理器

一个简洁优雅的提示词管理工具，帮助你组织和管理各种 AI 提示词。

## 功能特性

- 🗂️ **分类管理**: 自定义分类，支持添加图标
- 📝 **提示词管理**: 添加、编辑、删除提示词
- 📋 **一键复制**: 鼠标悬浮显示操作按钮，点击即可复制
- 💾 **本地存储**: 数据保存在浏览器本地，无需服务器
- 🎨 **简洁设计**: 白色简洁风格，优雅的悬浮效果
- 📱 **响应式设计**: 适配不同屏幕尺寸

## 开始使用

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 使用说明

1. **添加分类**: 点击左侧栏的 "+" 按钮添加新分类
2. **选择分类**: 点击左侧栏的分类名称查看该分类下的提示词
3. **添加提示词**: 选择分类后，点击右上角的"添加提示词"按钮
4. **复制提示词**: 鼠标悬浮在提示词上，点击复制按钮或直接点击内容区域
5. **编辑提示词**: 鼠标悬浮时点击编辑按钮
6. **删除提示词**: 鼠标悬浮时点击删除按钮
